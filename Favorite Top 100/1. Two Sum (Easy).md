# 1. Two Sum (Easy)

[LeetCode 題目連結](https://leetcode.com/problems/two-sum/)

## Ideas :
1. 使用 Hash Table（在 Python 中是 Dictionary）來記錄已經看過的數字與其索引。
2. 走訪陣列時，計算 `target - 當前數字`（也就是差額）。
3. 如果差額已經存在於字典中，代表找到答案，直接回傳兩者索引；若無，則將當前數字存入字典。

## Solution 1

Language : Python

```python
class Solution(object):
    def twoSum(self, nums, target):
        num_map = {}
        for i, num in enumerate(nums):
            diff = target - num
            if diff in num_map:
                return [num_map[diff], i]
            num_map[num] = i
